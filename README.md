Home Voyage is a home-finding demo app created to explore the usage of Typescript with React.

<br />

## Access Locally

<code>
In terminal, run git clone https://github.com/MightyJoeW/HomeVoyage.git

Install: `yarn`

Run app: `yarn start`

## Additional scripts

<code>
Run unit tests: `yarn test`
</code>
